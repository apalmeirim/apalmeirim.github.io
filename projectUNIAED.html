Algorithms and Data Structures Project

Project made for ADS (Algorithms and Data Structures).

Overview & Features:

    This project had the objective to create a system for managing a system of bedrooms that are available for rent for students that attend university. Initial JUnit testing, data structures, Maven, object-oriented programming, polymorphism, inheritance, and time complexity were some key concepts learned during the development of this project. Features that can be found are:

  • Adding, viewing and modifying rooms, managers and students.
  • Managing rooms dependent on students candidacys to a certain room, and occupying them respectively.
  • The whole project runs on the terminal using commands (which are visible on Main.java on Github).

    • The project can be found on Github below.

Tech Stack: Java
