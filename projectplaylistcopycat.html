Playlist Copycat

A simple app made to copy playlists from Spotify users directly to your own Spotify library.

Overview: This is a simple project that I created to practice creating a small web app. I used HTML and CSS for web development practice and used Python and Spotify's API to create the application.

Features:

  • Copies 100 songs at a time to new Spotify playlist, so limitless.
  • Other information about the playlist such as description and image cover is also copied.
  • Simply login to your Spotify account, copy and paste the URL of the playlist as many times as you want.
  • Project uploaded on Render, so redirect URI could make the app on Github unusable.

Tech Stack: Python, HTML and CSS
